# RedPy
（听说还挺好用的

## Install
```
pip install git+https://github.com/wangqixun/redpy.git@stable
```

或者 clone 下来手动安装

```
git clone -b stable https://github.com/wangqixun/redpy.git
cd redpy
pip install -e .
```
